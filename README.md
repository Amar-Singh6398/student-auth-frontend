# Student Authentication Frontend

Frontend for Student Exam Portal Authentication System.

## Features
- Login
- Register
- Forgot Password
- Reset Password
- API Integration
- Responsive UI

## Tech Stack
- React.js
- Tailwind CSS
- Axios
- React Router DOM

## Setup

1. Clone repository
git clone https://github.com/Amar-Singh6398/student-auth-frontend

2. Install dependencies
npm install

3. Start project
npm start

Frontend runs on:
http://localhost:3000
